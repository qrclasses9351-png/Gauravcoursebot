📘 GAURAV Utkarsh Downloader Bot (Webhook Mode for Render)

🚀 Render Setup (Webhook Mode)
------------------------------------
1️⃣ Upload these files (main.py, requirements.txt).
2️⃣ Environment Variable:
    BOT_TOKEN = your_telegram_bot_token
3️⃣ Start Command:
    python main.py
4️⃣ Deploy as a Web Service.

Render automatically provides RENDER_EXTERNAL_URL.
Bot automatically sets webhook to:
    https://your-app-name.onrender.com/webhook

✅ No polling conflict (no 409 error)
✅ Handles Hindi/encoded Utkarsh links
✅ Downloads & sends PDFs/videos directly
